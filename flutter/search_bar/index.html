Adding a Search Bar

Adding a Search Bar to the Infinite List from the previous tutorial.

In this tutorial I’ll add a search bar on top of the List View widget from the previous tutorial to provide search functionality using the posts' color keyword. Each Post has a color keyword that is used to request its random image. The user is able to search for posts through their color, as they type in the search bar.

Final Result:

To better organize the code, I created the static class PostServer which has n amount of random posts and acts as a server for our app. It has a single method: get(offset, limit, keywordFilter).

  • offset: int: from which index the client requires the data.
  • limit: int: how many results the client expects.
  • keywordFilter: String: what results the client wants.

Flutter TextField widget, with great customization support, was used to implement the search bar, and both the ListView and TextField are wrapped in a Stack widget to display one on top of the other.

The InfiniteList widget (from the previous tutorial) was updated with new properties to accomodate this functionality:

In a live scenario it’s not recommended to have every single change fetching data from the server, especially since most are not intended to trigger a new request. If so, the app would make at least one request per letter for a single word.

One way to avoid this is by adding a Timer with a fixed duration that executes the request function once it finishes. The timer resets every time the input changes.

We’re also adding a TextEditingController to access and clear the input field and a property _lastSearchTerm to store the input value used in the last request and avoid making a new one if it ends up being the same as the last.

The _search() function:

  • Resets the current timer, if active, and reassigns a new one to _timer.
  • It validates if it should make a new request by comparing the value used in the last request, once the timer finishes.
  • If so, it updates _postList with new data, passing the offset, the limit and the current input value.
